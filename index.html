HexPy, technically, is a family of tools for creating, modifying, and displaying 8-byte color hexcodes.

Non-technically, it's friggin awesome.

Currently HexPy is in Beta, accessible from a CLI only. Full code library coming soon.

This tutorial assumes you are familiar with a UNIX-based CLI and Python

Installation is easy:

  1. Have Python 3.4 or later installed. Note: on most Linux distros, you must install python3-tk as well
  2. Download and decompress the .zip or .tar from Github
  3. cd to the uncompressed directory
  4. Do chmod 777 pgexec
  5. Either add it to your PATH or run it with ./pgexec

To test HexPy:

  1. Run the ranhex.py script to generate a test image.
  2. From a terminal do pgexec imgtest.pgf You can also add a v at the end if you want to see live stats verbose error reports. However, this impacts performance

For more info, see the HexPy github repo. There's a link over there on the left.
