HexPy

Powerful Graphics. Extremely lightweight. Pure Python 3.

View on GitHub Download .zip Download .tar.gz

HexPy is a lightweight family of tools for creating, modifying, and displaying 8-byte color hexcodes. By using arrays of colorhexes to output graphics, a programmer has simple yet powerful access to the exact pixels being displayed onscreen.

Currently HexPy is in Beta, accessible from a CLI only. Full code library coming soon.

Installation is easy:

  1. Have Python 3 installed.
  2. Download the zip from Github
  3. Add pgexec to your PATH
  4. Done!

To test the script:

  1. Run the ranhex.py script to generate a test image.
  2. From a terminal do pgexec image.pgf (You can also add a 'v' at the end if you want to see verbose error reports.)

For more info see the HexPy github repo. Link above.
